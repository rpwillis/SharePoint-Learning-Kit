SLK/MLC SDK Documentation
Introduction

SharePoint Learning Kit (SLK) is a SCORM 2004-conformant e-learning delivery and tracking mechanism built as a SharePoint 3.0 solution. Microsoft Learning Components (MLC) is the foundation upon which SLK is built. The SLK/MLC SDK, which includes this documentation set, provides system integrators and software developers with the tools and information they need to:

  • customize and extend SLK, and
  • use MLC to add e-learning functionality to other their own applications.
