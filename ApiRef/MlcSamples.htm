Microsoft Learning Components
MLC Samples

The following samples demonstrate how to call the MLC API from applications.

Note  Although you can access the MLC API using any programming language compatible with the .NET common language runtime (CLR), the code samples in this documentation are all written in C#.

Sample Name Sample Type Description
ValidatePackage.cs Windows Forms Application Demonstrates how to use the MLC API to validate the contents of a SCORM 1.2, SCORM 2004 or Class Server LRM/IMS+ package.
BasicWebPlayer Web Application Sample Web application that executes e-learning content (in SCORM 1.2, SCORM 2004, or Class Server LRM/IMS+ format). Does not use or require SharePoint.
BasicWebPlayer includes the following files (among others):
BasicWebPlayerBase.aspx.cs Web Page Shows how to query LearningStore and convert the query results to HTML.
CreateAttempt.aspx.cs Web Page Demonstrates how to create an attempt on an e-learning package.
DeletePackages.aspx.cs Web Page Shows how to delete e-learning packages from a FileSystemPackageStore. Also shows how BasicWebPlayer finds and deletes all attempts on a package.
LStoreHelper.cs Web Page Helper methods for casting an object returned from LearningStore queries to the correct type.
PageHelper.cs Web Page Shows how to add users to the UserItem table in the MLC database, and how to obtain the current user's UserItemIdentifier and other information from UserItem.
SequencingLog.aspx.cs Web Page Demonstrates how to read the sequencing log for an attempt. This log can help content authors understand how MLC is interpreting SCORM sequencing and navigation rules in their content.
UploadPackage.aspx.cs Web Page Shows how to add a package to a FileSystemPackageStore. Also shows how BasicWebPlayer maintains additional information about packages in extra columns in the PackageItem database table. These extra columns are defined in BasicWebPlayer's Schema.xml file.
Schema.xml LearningStore Schema XML Extends the MLC base schema with database extensions specific to BasicWebPlayer.
